`IWrap` is a wrap widget with a cleaner (but equivalent) layout implementation and an 'insertionIndexAt' method for drag and drop insertion.

(it was forked out from [animated_containers](https://github.com/makoConstruct/animated_containers) after mako discovered animated_to, reflected on it, and decided that yeah the container shouldn't be the part that handles animation, so let's have a version of that with the animation stuff stripped out)
